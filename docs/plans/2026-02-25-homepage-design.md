# Дизайн главной страницы — клон santehnika-online.ru

## Цель

Создать главную страницу интернет-магазина сантехники, воспроизводящую структуру, блоки и визуальный стиль santehnika-online.ru. Главная — навигационная страница с промо-контентом, подборками товаров и категориями.

## Структура страницы (14 секций, сверху вниз)

```
 1. Header (существует)
 2. Hero: промо-слайдер + товар дня (авто-компонент шаблона)
 3. Иконки категорий (горизонтальный ряд)
 4. Промо-баннеры (3 карточки в ряд)
 5. Подборка по категориям (табы + карточки товаров)
 6. Бренды (логотипы в ряд)
 7. Большая акция + карусель товаров (баннер слева, товары справа)
 8. Сервисные карточки (Рассрочка · Установка · Доставка)
 9. Готовые решения (карусель комплектов)
10. Все акции (сетка промо-баннеров)
11. Ещё карусель товаров
12. Идеи для дома (masonry-галерея фото)
13. Наши преимущества (иконки + текст)
14. Футер (3 колонки ссылок + контакты)
```

## Новые коллекции Payload

### Promotions

Промо-баннеры, переиспользуемые на разных страницах (hero-слайдер, промо-сетки, акции).

| Поле | Тип | Описание |
|------|-----|----------|
| title | text | Заголовок баннера |
| subtitle | text (опц.) | Подзаголовок |
| image | upload(media) | Фон/изображение баннера |
| link | link | Куда ведёт |
| type | select: hero / banner / sale | Тип промо (влияет на размер/стиль) |
| active | checkbox | Активен ли |
| categories | relationship(categories, hasMany) | Привязка к категориям (пусто = глобальный) |

### FeaturedProducts

Товары дня — компактная карточка со скидкой.

| Поле | Тип | Описание |
|------|-----|----------|
| title | text | Заголовок (напр. «Товар дня») |
| product | relationship(products) | Конкретный товар |
| discountPercent | number | Процент скидки |
| active | checkbox | Активен ли |
| categories | relationship(categories, hasMany) | Привязка (пусто = глобальный) |

### Brands

Бренды с логотипами.

| Поле | Тип | Описание |
|------|-----|----------|
| name | text | Название бренда |
| logo | upload(media) | Логотип |
| link | link (опц.) | Ссылка на страницу бренда |
| sortOrder | number | Порядок отображения |

## Новые блоки layout builder

### CategoryIconsRow

Горизонтальный прокручиваемый ряд корневых категорий с маленькими иконками.

- Поля: `categories` (relationship, hasMany) или автоматически все корневые
- Фронтенд: горизонтальный скролл, иконка 48-64px + название

### PromoBanners

Ряд из 3-4 промо-карточек.

- Поля: `source` (all_active | manual), `promotions` (relationship Promotions, hasMany)
- Фронтенд: CSS Grid 3 колонки, aspect-ratio ~16:9, изображение + заголовок

### CategoryProductTabs

Табы категорий с карточками товаров выбранной категории.

- Поля: `categories` (relationship, hasMany — какие табы показывать), `limit` (number, дефолт 4)
- Фронтенд: горизонтальные табы, при переключении — карточки товаров. Клиентский компонент с fetch.

### BrandsBlock

Логотипы брендов.

- Поля: `title` (text), `brands` (relationship Brands, hasMany) или автоматически все
- Фронтенд: сетка логотипов 6-8 в ряд, ссылка «Все бренды»

### SaleWithCarousel

Большой промо-баннер слева + карусель товаров справа.

- Поля: `promotion` (relationship Promotions — один баннер), `source` (by_category | manual), `category`, `products`
- Фронтенд: 2 колонки (40/60), баннер слева, embla-carousel товаров справа

### ServiceCards

3 информационных карточки (Рассрочка, Установка, Доставка).

- Поля: `cards` (array: icon(text — имя Lucide-иконки), title, description, link)
- Фронтенд: 3 колонки, иконка + заголовок + описание

### ProductSets

Готовые решения — карусель комплектов.

- Поля: `title` (text), `sets` (array: title, image, link, price) или relationship на товары
- Фронтенд: горизонтальная карусель карточек

### PromoGrid

Сетка промо-баннеров (3 штуки).

- Поля: `promotions` (relationship Promotions, hasMany, max 4)
- Фронтенд: CSS Grid, 3 карточки с изображениями

### ImageGallery

Masonry-галерея фото (идеи для дома).

- Поля: `title` (text), `images` (array: image(upload media), caption(text опц.))
- Фронтенд: CSS masonry layout (columns) или CSS Grid с разными span

## Авто-компонент Hero

Рендерится шаблоном страницы перед layout-блоками. Не является layout block.

- Тянет данные из `Promotions` (type=hero, active=true) + `FeaturedProducts` (active=true)
- Для главной: все глобальные (без привязки к категории)
- Для страниц категорий: фильтрует по категории
- Layout: промо-слайдер слева (~70%), карточка товара дня справа (~30%)

## Футер

Переработать компонент Footer. Источник данных — Payload Global `footer`.

Структура:
- 3 колонки ссылок: О компании, Каталог, Покупателям
- Контакты: телефон, email
- Копирайт
